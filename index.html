GlitchUp

2018

GlitchUp is a macOS app that allows you to apply glitch effects to images. Instead of just being a set of glitch filters, this software has a couple of intentionally kept bugs, allowing experimentation. By activating the 'superglitch' mode, you will trigger a bug in the GlitchUp algorithm, enabling you to create unexpected visual results (effects piling up, cancel doesn't really cancel the effect...). Published on CNET and itch.io
